HeavyProcessKiller
HeavyProcessKiller is a utility script designed to identify and terminate heavy processes consuming excessive system resources. This tool is particularly useful for maintaining optimal system performance and preventing resource hogging by rogue processes.

Features
Identifies heavy processes based on CPU and memory usage thresholds.
Provides options to list heavy processes before termination.
Allows customization of thresholds for CPU and memory usage.
Offers safe termination with confirmation prompts.
